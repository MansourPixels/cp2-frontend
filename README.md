# 🍔 Redesign — Lanches Mansour

Projeto de entrega do Checkpoint 2 da disciplina Front-end Design, ministrada pela professora Cynthia Sayuri.

Este repositório contém o redesign do site institucional da Lanches Mansour, tradicional lanchonete paulistana conhecida por hambúrgueres artesanais e maionese de fabricação própria.

---

## Contexto do grupo

A Mansour vai além da escolha de marca para o trabalho: foi no próprio estabelecimento que o grupo se encontrou pela primeira vez depois da revelação do challenge da Jovi. Esse lugar marca o início da trajetória em conjunto — e por isso o projeto carrega um significado especial para todos os integrantes.

---

## O que foi construído

Landing page em página única com navegação interna, destacando:

- **Hero** — apresentação da marca, call-to-action para delivery (iFood) e link para o cardápio  
- **Sobre nós** — história e valores da lanchonete  
- **Cardápio** — destaque dos produtos  
- **Unidades** — Cambuci e Aclimação, com galeria expansível e **carrossel de fotos** por unidade
- **Contato** — canal de comunicação com o cliente  
- **Rodapé** — encerramento da página  

Visual pensado em **tons escuros e dourados**, tipografia **Montserrat** e **Playfair Display** (Google Fonts), alinhado a uma identidade mais atual sem perder a sensação de tradição.

---

## Como executar localmente

1. Clone ou baixe este repositório.  
2. Abra o arquivo **`index.html`** no navegador **ou** use uma extensão tipo **Live Server** no VS Code apontando para a pasta do projeto.
